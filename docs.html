W A S D

Reset property sliders:

PressL

Place blocks on top of each other:

Right-click

Middle-click (mouse wheel)

Hold Shift

Ctrl + P

KL Script - Basics

Update player position

Sets the x or y position of the player

Syntax: 

game.local.player.vector2.x or
game.local.player.vector2.y
       number or
       getVarValue

Example:

This example sets the player's x-position to 5. 

game.local.player.vector2.x
       5

Create a variable

Creates a new variable with the name 
variable_name

Syntax: 

game.local.var+
       variable_name
       variable_starting_value (only numbers)

Example:

This example creates a variable named test with value 19 

game.local.var+
       test
       19

Set a variable to something

Sets the value of the variable with the name 
variable_name

Syntax: 

game.local.var=
       variable_name
       value/getVarValue/game.local.player.vector2.(x or y)

Example:

This example sets the variable named test to 19 

game.local.var=
       test
       19

Add to a variable

Adds something to the variable called 
variable_name

Syntax: 

game.local.var.addToVar=
       variable_name
       value/getVarValue/game.local.player.vector2.(x or y)

Example:

This example adds 19 to the variable called test 

game.local.var.addToVar=
       test
       19

Stop the script for an amount of seconds

Makes the script wait 
seconds

Syntax: 

game.local.time.wait=
seconds

Example:

This example makes the script wait 1.5 seconds 

game.local.time.wait=
1
