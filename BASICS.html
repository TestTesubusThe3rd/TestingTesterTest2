Basic controls

Move camera: W A S D

Reset property sliders: press L

Place blocks on top of each other: right cilck

Copy blocks properties: middle click (mouse wheel)

Place blocks off grid: hold shift

Shortcuts

Publish game: Ctrl + P NOTE: If you put the same name as an already uploaded game, you will update that game!

Change time of day: M
